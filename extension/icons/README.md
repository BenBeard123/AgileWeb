# Extension Icons

Place your extension icons here:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

You can create these using any image editor or online icon generator. A simple shield icon works well for AgileWeb.

For now, the extension will work without these icons, but Chrome will show a default icon.
